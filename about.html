The programming languages I use

HTML

HTML (HyperText Markup Language) is the core structure of every website. I use it to build clean, semantic, and accessible layouts that form the foundation of responsive web interfaces.

CSS

CSS (Cascading Style Sheets) allows me to transform structure into polished, modern, and responsive designs. I use CSS to create layouts, animations, components, and consistent visual systems across projects.

Javascript

JavaScript brings websites to life. I use JS to build interactive interfaces, handle dynamic content, perform API calls, manage state, and create smooth, user-friendly functionality across all devices.

Java

Java is a powerful, object-oriented language used for backend systems, Android development, and enterprise applications. I use Java to build structured, scalable solutions with strong performance and reliability.

C++

C++ is a high-performance, low-level programming language used in system development, game engines, and performance-critical applications. It strengthened my understanding of memory management, data structures, and algorithm optimization.

C#

C# is a versatile programming language widely used for desktop apps, enterprise systems, and Unity game development. I use C# for building clean, structured applications and understanding modern OOP practices.

SQL

SQL is essential for managing and querying databases. I use SQL to design tables, optimize queries, build relationships, and ensure efficient data retrieval for applications and backend services.

Python

Python is a flexible, beginner-friendly language used in automation, scripting, machine learning, and backend development. I use Python to build utilities, data-processing scripts, and quick prototypes.

Kotlin

Kotlin is a modern, expressive programming language designed for Android development and cross-platform applications. I use Kotlin to build clean, efficient, and well-structured mobile apps with improved safety, reduced boilerplate, and seamless compatibility with existing Java code.
